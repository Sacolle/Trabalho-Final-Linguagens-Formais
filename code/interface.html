Interface do Autômato

Explicação

  • Método de Utilização:
    1. Coloque a definição do AFD na caixa Autômato.
    2. Pressione o botão Processar.
    3. Coloque a(s) palavra(s) desejada(s) na caixa Palavras de Entrada.
    4. Pressione o botão Reconhecer.
  • Observações:
    1. Autômatos devem seguir as especificações do enunciado.
    2. Após o passo 2, Estatus do Autômato deve ficar: Inicializado.
    3. Após o passo 2, a caixa Autômato conterá a versão mínima do autômato de entrada.
    4. Palavras devem ser separadas por quebra de linha.
    5. Palavras serão divididas entre ACEITAS e REJEITAS.
